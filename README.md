# Kur-Takip

**Kur-Takip**, kullanıcıların anlık döviz hesaplaması yapmasını ve döviz kurlarını takip etmesini sağlayan basit bir döviz uygulamasıdır.

## Özellikler

- Anlık döviz kurları görüntüleme  
- Farklı para birimleri arasında hızlı hesaplama  
- Basit ve kullanıcı dostu arayüz (Shadcn UI ile tasarlanmış)  
- Gerçek zamanlı veriler: [Frankfurter API](https://www.frankfurter.app/) kullanılarak sağlanıyor  

## Teknolojiler

- **Next.js** – React tabanlı framework  
- **Tailwind CSS** – Hızlı ve özelleştirilebilir CSS  
- **Shadcn UI** – Modern UI bileşenleri  
- **Frankfurter API** – Döviz verileri için ücretsiz API

3. Gerekli paketleri yükleyin:
